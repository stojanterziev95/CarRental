# Car Rental Project
Built with Java 17 and Spring Boot 2.7.2

## Installation
- Clone the repository
- Import the project into IntelliJ IDEA
- Run the application

## Database
- Create a database named `car_rental_project`


## Prerequisites
- Java 17
- Maven 3.8.5
- MySQL 8.0.29
- IntelliJ IDEA 2022.2.1 (Community Edition)
